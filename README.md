# 🛍️ Retail Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes retail sales data using **Python, SQL, and Power BI**. It demonstrates the complete data analytics workflow from data cleaning to interactive dashboard creation.

## 🚀 Tools & Technologies
- Python
- Pandas
- NumPy
- MySQL
- Power BI
- Jupyter Notebook

## 📊 Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Total Customers KPI
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Top 10 Products
- Top 10 Customers
- Interactive Slicers

## 🔍 Business Insights
- Identified top-performing product categories.
- Compared regional profit performance.
- Analyzed monthly sales trends.
- Found the highest-value customers.
- Identified the best-selling products.
